# 글로벌 프로젝트 규칙

## 1. 코드 스타일
- 모든 코드는 TypeScript로 작성
- ESLint와 Prettier 설정 필수
- 파일명은 kebab-case 사용
- 클래스명은 PascalCase 사용
- 변수/함수명은 camelCase 사용
- 상수는 UPPER_SNAKE_CASE 사용
